  Vrinda Store Data Analysis (Excel Project)

 📌 Project Overview
This project analyzes sales data of Vrinda Store using Microsoft Excel to uncover business insights, customer behavior, and sales trends.

🎯 Objectives
- Analyze sales performance across months
- Understand customer demographics (age, gender)
- Identify top-selling categories and channels
- Generate insights for business growth

🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning
 📂 Dataset
- Dataset Name: Vrinda Store Sales Data
- Includes:
  - Order ID, Customer ID
  - Gender, Age, Age Group
  - Month & Date
  - Sales Amount
  - Product Category & Size
  - Sales Channel (Amazon, Flipkart, etc.)
  - Location (City, State)

📊 Key Analysis Performed
- Monthly Sales Trend Analysis
- Gender-wise and Age-wise Sales
- Channel-wise Performance
- Top Categories & Products
- Regional Sales Analysis
 📈 Key Insights
- Majority of customers belong to specific age groups (e.g., adults)
- Certain months show higher sales (seasonal trends)
- Online channels contribute most to revenue
- Specific categories generate maximum sales
